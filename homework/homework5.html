Task 1:

I have created a Java program which you must amend, install and run on your machine. You may need to use other resources given above to help you do this.

First you will find a general discussion about the anatomy of Java programs using an example called Sample.java - you can then save and edit this program - along with another program (RunSample.java) which you will need to run the first program. Read the program notes very carefully before attempting the practical part of this homework

Here is what you must achieve:

  • Edit the Sample.java program so that the label (jLabel1) shows your group number (e.g. 99).
  • Edit the RunSample.java program so that it runs with your group identity (e.g. y2006g99).
  • Run the program and take a snapshot of the window it produces so that your group number can be clearly seen.
  • Take a snapshot of the command window showing a prompt with your name.
  • Put the two images on your web site as homework for week 5.
  • You need to replace the BufferedReader with a Scanner and link your code on your web page
