1. Use the following command:

while(list($key, $val) = each($row)){

}

3. The differences are reflected in the following points:

  • PHP programs are usually embedded in the HTML page, while CGI programs are always separate programs.
  • b. The filename extension of PHP programs are often .php, .php3 or .phtml, while those of CGI programs are diverse, which depend on the language. That means there are many language can be used to write a CGI program.
  • c. PHP processor is one integrated part of Web Server, while CGI programs are usually compiled or interpreted by their own tools.
